# Couple Diary Web App

커플의 기념일(D-day)과 데이트 기록을 간단하게 관리하기 위한  
Vanilla JavaScript 기반 웹 애플리케이션입니다.

GitHub Pages를 통해 배포되었으며,  
브라우저 로컬 스토리지를 활용해 별도의 서버 없이 동작합니다.

## Demo
👉 https://mymelo-coder.github.io/couple-diary/

## Features
- 사귄 날 입력 및 D-day 자동 계산
- LocalStorage 기반 데이터 저장
- 새로고침 후에도 데이터 유지
- 모달 UI를 활용한 사용자 인터랙션
- GitHub Pages를 통한 정적 웹 배포

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- GitHub Pages

## What I Learned
- DOM 이벤트 처리 및 상태 관리
- 브라우저 저장소(LocalStorage) 활용 방법
- 디버깅을 통한 오류 원인 분석 경험
- Git/GitHub를 이용한 버전 관리 및 배포 흐름 이해
